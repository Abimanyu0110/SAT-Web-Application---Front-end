# Step 1:

-> Make sure your node version is 20+
-> If no means upgrade to the required version 20

# Step 2:

-> Add new file in the root folder with this name --> .env.local 
-> Add this line in the .env.local file --> VITE_API_HOST=http://localhost:8080

## Step 3:

-> Make sure you're in the root folder or else use cd to get into root folder.
-> Next run this command in the terminal --> npm i --force
-> The above command will install the necessary node modules.

## Step 4:

-> Once all the node molules were installed, use this command to run this project --> npm run dev
-> Now you get a localhost port, Click that or copy that link and open it in the Browser
-> Now your front end is ready, Now go and run Backend.
